# Commas Rule

Controls the number of spaces before and after commas (`,`).

## Options

- `max-spaces-before`: Maximum spaces allowed before commas (use -1 to disable)
- `min-spaces-after`: Minimum spaces required after commas
- `max-spaces-after`: Maximum spaces allowed after commas (use -1 to disable)

## Default Configuration

```yaml
rules:
  commas:
    max-spaces-before: 0
    min-spaces-after: 1
    max-spaces-after: 1
```

## Special Configuration

This rule is enabled by default with strict spacing requirements.
