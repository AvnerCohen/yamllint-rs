# Colons Rule

Controls the number of spaces before and after colons (`:`).

## Options

- `max-spaces-before`: Maximum spaces allowed before colons (use -1 to disable)
- `max-spaces-after`: Maximum spaces allowed after colons (use -1 to disable)

## Default Configuration

```yaml
rules:
  colons:
    max-spaces-before: 0
    max-spaces-after: 1
```

## Special Configuration

This rule is enabled by default with strict spacing requirements.
